Detailed Description

Klasa odpowidająca za odczyt i komunikację z programem.

Ta klasa umożliwia korzystanie z aplikacji, wczytywanie mapy, zwizualizowanie jej na ekranie oraz operowanie na elementach wczytanej mapy. Wykorzystuje klasy Algorithm() oraz List().

◆ loadMap()

void MainWindow::loadMap ( )

Funkcja wczytująca mapę do programu.

Funkcja przetwarza tylko pliki o rozszerzeniu .map, przy czym zawartość tych plików musi zawierać informacje o wysokości oraz szerokości mapy w pierwszych 7 linijkach tekstu. Od 7-mej linijki funkcja tworzy listę list (których ilość odpowiada wysokości mapy), a każda kolejna lista począwszy od początkowej aż do końcowej zawiera w sumie tyle elementów, ile określa wartość szerokości mapy. Wartości dla domyślnych map wynoszą wysokość = 512 oraz szerokość = 512, i dla takich rozmiarów mapy funkcja będzie działać prawidłowo. Wczytana mapa jest nastepnie wizualizowana przy pomocy pixmapy. Elementy, po których można się poruszać, zaznaczone są na szaro, natomiast przeszkody przedstawione są na czarno.
