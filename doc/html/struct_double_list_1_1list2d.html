Detailed Description

Struktura danych przedstawiająca dwukierunkową listę dwukierunkowych list.

Struktura zawiera konstruktor oraz takie informacje, jak: wielkość listy list, wskażnik na pierwszą/ostatnią listę oraz funkcje służące do manipulacji zawartością listy list.
