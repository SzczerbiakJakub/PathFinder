Detailed Description

Struktura danych przedstawiająca listę dwukierunkową.

Struktura zawiera konstruktor oraz takie informacje, jak: wielkość listy, wskażnik na pierwszy/ostatni element listy, wskaźnik na poprzednią/następną w kolejności listę oraz funkcje służące do manipulacji zawartością listy.
